Bem-vindo ao Meu Bot do Discord!

Este é o meu bot para Discord, criado para facilitar a interação em servidores.

Comandos do Bot

Veja os comandos disponíveis para usar com o bot!

-olá

O bot responde com uma saudação.

-tempo

Mostra o tempo que o bot está online.

-timer <data> <hora>

Define um timer para uma data e hora específicas.

-expulsar @usuario

Expulsa um usuário do servidor (necessário permissão de expulsão).

-banir @usuario

Bane um usuário do servidor (necessário permissão de banimento).

-menu

Exibe todos os comandos disponíveis.

-userinfo

Exibe informações sobre o usuário que enviou o comando.

-ping

Verifica a latência do bot.

-serverinfo

Mostra informações sobre o servidor.

-avatar @usuario

Exibe o avatar de um usuário específico.

-say <mensagem>

Faz o bot enviar uma mensagem em nome do usuário.
